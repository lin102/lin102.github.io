The Trace of Elephants

Description

This work tells a story about Elephant Protection in an interactive way. This way of story-telling is inspired by Esri Story Maps.
The initial idea of this work is about to make trials on different ways of user interaction in web mapping and web cartography. I combine the text-scrolling with base map-changing Simultaneously while the user reading the page downward. Another attempt is that drawing maps using SVG technic, which can display sharp vector graphics and furthermore provide ways of user interaction, in this case(see in Elephants Distribution), hover. This webpage is also responsive designed.
I created this work using Leaflet.js, illustrator.
